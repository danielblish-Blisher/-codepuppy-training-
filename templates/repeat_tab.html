🔄 Repeat Work Orders Analysis

Track work orders that have been repeated or recalled at the same location

ℹ️ Data Source & Methodology

📡 Data Source

Aggregated from BigQuery re-crystal-mdm-prod.crystal.sc_workorder — all FM trades, completed in the last 12 months (routine PM / Scheduled Service excluded).

🔄 What Counts as a “Repeat”?

A pattern is flagged as a repeat when the same store had the same problem type completed more than once within 30 days in the last 12 months. Specifically:

  1. All FM completed WOs are grouped by Store # + Problem Type using a LAG window function in BigQuery.
  2. Any pair with a consecutive completion within 30 days is flagged as a repeat.
  3. Each row in this view represents one store+problem pair with repeat history. Repeats (30d) counts how many consecutive completions were within 30 days.

📊 Key Metrics Explained

  • Store+Problem Pairs — Unique store+problem combinations with at least one 30-day repeat in the last 12 months.
  • 30-Day Repeat Incidents — Total count of consecutive completions within 30 days across all pairs.
  • Stores Affected — Number of unique store locations with at least one repeat pattern.
  • Unique Problem Types — Distinct problem types appearing in repeat patterns.

🔍 Views

  • By Problem Type — Groups repeats by problem category to show which issue types recur most.
  • By Store — Shows which stores have the most repeat visits.
  • By Director — Aggregates repeat patterns by FM Director for leadership visibility.
  • Detail List — Full list of individual repeat work orders with sortable columns.
Note: Data refreshes when the dashboard syncs with BigQuery. Org hierarchy (SR Director, FM Director, Regional, FS Manager) is enriched from the stores dataset. Filters apply across all views.

ℹ️ How to Read This:

  • Repeats (30d) = consecutive completions of the same problem at the same store within 30 days — higher = more problematic
  • By Problem Type shows which FM issue categories keep recurring across all stores
  • By Store shows which locations have the most chronic repeat issues
  • By Director shows area responsibility and top repeat territories
  • Detail List shows every store+problem pair — sortable by repeat count, NTE, or last seen date
